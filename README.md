# 🌟 My Ubuntu i3wm Dotfiles

Welcome to my personal Linux rice! This repository contains my highly customized configuration files for a clean, minimalist, and productive workflow on Ubuntu using the **i3 Window Manager** and the **Catppuccin Mocha** color palette.

---

## 🎨 Preview & Aesthetics
* **Color Scheme:** Catppuccin Mocha ☕
* **Window Manager:** i3wm
* **Bar:** Polybar (Status bar showing system stats)
* **Launcher:** Rofi (Minimalist application menu)
* **Terminal:** Alacritty
* **Compositor:** Picom (For smooth animations and transparency)

---

## 📂 Repository Structure

The core configuration files are organized as follows:

```text
.
├── alacritty/       # Terminal styles and color palette
├── btop/            # System monitor theme
├── cava/            # Audio visualizer configurations
├── dunst/           # Minimalist notification styling
├── i3/              # Keybindings, workspace rules, and layout
├── picom/           # Compositor settings for shadows & opacity
├── polybar/         # Top status bar configurations and launch script
├── rofi/            # Application launcher themes
└── background       # Current desktop wallpaper
